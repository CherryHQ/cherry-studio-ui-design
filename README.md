# Cherry Studio UI Design

Cherry Studio V2 的产品设计与 UI 交互原型。基于 React + TailwindCSS + shadcn/ui 构建的可交互设计稿。

## 快速开始

```bash
pnpm install
pnpm dev
```

## 技术栈

- React 18 + TypeScript
- TailwindCSS v4
- shadcn/ui (Radix UI)
- Framer Motion
- Vite

## 项目结构

```
src/
  app/              # 应用入口、布局、路由、上下文
  features/         # 功能页面（chat、settings、extensions、knowledge 等）
  styles/           # 主题 CSS 变量
```
